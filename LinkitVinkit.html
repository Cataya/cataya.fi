Vinkkejä

 

Vinkki 9, Linux:

Tuleeko päivitysilmoitus aina väärään hetkeen?
Voit päivittää Linuxin terminaalista manuaalisesti:
Ubuntu: sudo apt update && sudo apt upgrade -y (kysyy salasanaa)
Arch: sudo pacman -Syu -y

RedHat: sudo dnf update

 

Vinkki 8, Linux:

Kun liität komentoja terminaaliin niin se ei toimikkaan kuten normaalisti ctrl + c tai ctrl + v
Käytä sen sijaan ctrl + shift + c ja ctrl + shift + v

 

Vinkki 7, Windows:

Kun liität esim. testiä, jossa on ihan eri muotoilu kuin paikassa, johon se pitäisi liittää
Kun liität teksitin, käytä ctrl + v tilalla ctrl + shift + v,
niin teksti ottaa muotoilut jo olemassa olevasta asiakirjasta.

 

Vinkki 6, Windows:

Leikepöydälle voit kopioida useita asioita samaan aikaan:
Paina ensin Windows-näppäin + v-näppäin ja ota tämä toiminto käyttöön.
Jatkossa löydät samalla näppäin yhdistelmällä 25 viimeisintä kopioimaasi asiaa.
Voit myös Pinnata listaan useiten käyttämiäsi asioita.

 

Vinkki 5, Gmail:

Gmailista voi hakea isoja liitetiedostoja, jos esim. tila alkaa käytäämään vähiin
Kirjoita hakukenttään: has:attachment lager:20m Gmail näyttää sähköpostit, joissa liitetiedostot on isompia kuin 20megaa.

 

Vinkki 4, Gmail:

Gmailista voi lähettää salatun sähköpostin seuraavasti:
Kun olet kirjoittanut sähköpostin, paina alalaidassa olevaa lukon kuvaa, jonka päällä on kello. Vastaa annettuihin kysymyksiin.
Viestin voimassaoloaika, salasana tekstiviestillä(SMS) ja viestin lähetysvaiheessa kysytään vastaanottajan puhelinnumeroa.

 

Vinkki 3 Windows:

Windows10 koneen sammutus jättää koneen valmius-tilaan. Pidä Shift-näppäintä pohjassa
kun sammutat koneen, niin kone samuu oikeasti.

 

Vinkki 2, Windows:

Tietystä alueesta saa kuvakaappauksen painamalla ensin näppäinyhdistelmää Windows-näppäin + Vaihto + S,
niin osoitin muuttuu ristiosoittimeksi. Valitse sitten haluamasi alue hiirellä vetämällä.
